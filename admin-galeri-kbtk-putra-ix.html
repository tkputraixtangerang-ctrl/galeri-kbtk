📌 Cara Upload Foto Baru ke Website
  1. Buka Google Drive sekolah, masuk ke folder "Galeri KB-TK Putra IX"
  2. Klik "+ Baru" → "Upload file" → pilih foto dari HP/laptop
  3. Tunggu sampai upload selesai, lalu klik kanan foto tersebut
  4. Pilih "Bagikan" → ubah ke "Siapapun yang memiliki tautan" → klik "Salin tautan"
  5. Buka file Admin Panel ini, masuk tab "Tambah Foto"
  6. Paste link di kolom pertama → isi judul, deskripsi, kategori → klik "Tambah ke Daftar"
  7. Masuk tab "Generate Kode" → klik "Generate Hanya Foto Baru" → salin kode
  8. Kirim kode via WhatsApp ke pengelola website
✏️ Cara Edit Judul/Deskripsi Foto
  1. Masuk ke tab "Kelola Daftar"
  2. Cari foto yang ingin diedit
  3. Klik tombol "✏️ Edit" (kuning)
  4. Ubah judul atau deskripsi → klik "Simpan"
  5. Setelah edit, pergi ke tab "Generate Kode" → generate ulang → kirim ke pengelola
🗑️ Cara Hapus Foto dari Galeri
  1. Masuk ke tab "Kelola Daftar"
  2. Cari foto yang ingin dihapus
  3. Klik tombol "🗑️ Hapus" (merah) → konfirmasi
  4. Lanjut ke tab "Generate Kode" → generate ulang semua → kirim ke pengelola
⚠️ Hal Penting yang Perlu Diingat
  • Data di panel ini tersimpan di browser. Jangan hapus cache atau gunakan browser berbeda.
  • Selalu backup data dengan klik "Generate Kode" dan simpan kodenya.
  • Foto di Google Drive harus diatur "Siapapun yang punya link" agar tampil di website.
  • Ukuran foto ideal: 800x600px ke atas, format JPG atau PNG.
  • Setelah generate kode, kirim ke pengelola website dalam 24 jam agar tidak lupa.
